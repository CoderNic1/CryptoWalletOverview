# Stake Ledger

A single-file, fully offline web app to track where you stake tokens or NFTs:
network, wallet, project links (Discord/X/website), founders, notes, and an
activity log per project (stake / unstake / claim / note).

## Run it
Just open `index.html` in a browser, or host it as a static site (e.g. GitHub Pages).

## Data & privacy
- All data is stored in your browser's `localStorage`. Nothing is sent to any server.
- Use **Export data (.txt)** regularly to back up — a JSON file you can re-import.
- **Import** supports merge or full replace.
- **Remove all data** wipes localStorage for this app permanently.
- Never enter private keys or seed phrases — public info only.

## Hosting on GitHub Pages
1. Push this folder to a repo.
2. Settings → Pages → set source to your branch / root.
3. Done — `index.html` is served as the site.
